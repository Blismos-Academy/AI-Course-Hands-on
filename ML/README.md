
# 🤖 Machine Learning Hands-On

This hands-on contains beginner-friendly Machine Learning examples implemented using Python and Scikit-learn.

## 📂 Folder Structure

```
ML/
├── Datasets/
│   ├── Disease_Prediction_Dataset.csv
│   ├── HousePrice.csv
│   └── Mall_Customer_Dataset_INR.csv
│
├── ipynb/
│   └── Machine learning.ipynb
│
└── README.md
```

## 📘 Topics Covered

### 🏠 House Price Prediction
- **Algorithm:** Simple Linear Regression
- **Dataset:** `Datasets/HousePrice.csv`

### 🏘️ House Price Prediction (Multiple Features)
- **Algorithm:** Multiple Linear Regression
- **Dataset:** `Datasets/HousePrice.csv`

### 🩺 Disease Prediction
- **Algorithm:** Logistic Regression
- **Dataset:** `Datasets/Disease_Prediction_Dataset.csv`

### 🛍️ Customer Segmentation
- **Algorithm:** K-Means Clustering
- **Dataset:** `Datasets/Mall_Customer_Dataset_INR.csv`

## 🛠️ Requirements

- Python 3.x
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## ▶️ Getting Started

Install the required packages:

```bash
pip install pandas numpy matplotlib scikit-learn notebook
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```

Open:

```
ipynb/Machine learning.ipynb
```

Run the notebook cells in sequence to follow the complete hands-on.

Happy Learning! 🚀
