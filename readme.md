# Machine Learning Hands-On

This HandsOn contains beginner-friendly Machine Learning examples implemented using Python and Scikit-learn.

## Repository Structure

```
ML-HANDSON/
│
├── Datasets/
│   ├── Disease_Prediction_Dataset.csv
│   ├── HousePrice.csv
│   └── Mall_Customer_Dataset_INR.csv
│
├── ipynb/
│   └── Machine learning.ipynb
│
└── README.md
```

## 📘 Topics Covered

### 📈 Simple Linear Regression
- **Purpose:** Predict house prices using a single feature
- **Dataset:** `Datasets/HousePrice.csv`

### 🏡 Multiple Linear Regression
- **Purpose:** Predict house prices using multiple features
- **Dataset:** `Datasets/HousePrice.csv`

### 🩺 Logistic Regression
- **Purpose:** Predict disease based on patient symptoms
- **Dataset:** `Datasets/Disease_Prediction_Dataset.csv`

### 🛍️ K-Means Clustering
- **Purpose:** Segment customers based on income and spending habits
- **Dataset:** `Datasets/Mall_Customer_Dataset_INR.csv`

### 🔀 Train-Test Split
- **Purpose:** Split the dataset into training and testing sets before model training
- **Dataset:** `Datasets/HousePrice.csv`

## Requirements

- Python 3.x
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Run

```bash
pip install pandas numpy matplotlib scikit-learn notebook

jupyter notebook
```

Open:

```
ipynb/Machine learning.ipynb
```

Run the notebook cells in order.
